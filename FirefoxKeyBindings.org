** Navigation
Alt + arrowkey   -> forward backward

Ctrl + r   -> reload page

Open file  ->  Cntrl-o

Cntrl + D bookmark this page

Cntrl + U    page source  -> this gives the source code for page and is !Awsome!

Cntrl + L   -> select location bar
Alt + D   ----^

Cntrl + T   new tab
